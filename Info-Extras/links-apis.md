# Links e Contatos – Portais Imobiliários e Integrações (Brasil)

> Objetivo: concentrar, em um único lugar, os principais **links oficiais** de portais imobiliários brasileiros e pontos de contato para **integração via API/feeds** e uso em CRM/omnichannel.

---

## 1. Grupo OLX – ZAP Imóveis, Viva Real e OLX

### 1.1. Portais
- ZAP Imóveis: https://www.zapimoveis.com.br
- Viva Real: https://www.vivareal.com.br
- OLX Imóveis: https://www.olx.com.br/imoveis

### 1.2. Integração / API / Feeds
- Portal de Integração (Grupo ZAP / Grupo OLX):  
  https://developers.grupozap.com

  Normalmente usado para:
  - Envio de anúncios via **feed XML**
  - Atualização de estoque de imóveis
  - Webhooks de leads (contatos gerados nos portais)

### 1.3. Onde falar sobre integração
Como política de privacidade e contatos podem mudar, o caminho mais seguro é sempre:

1. Acessar a área de ajuda/ajuda técnica do portal (ex.: “Ajuda” ou “Central de Ajuda” no ZAP/Viva Real).
2. Procurar por termos como **“integração”**, **“feed XML”** ou **“API”**.
3. Usar o formulário de suporte comercial/técnico informando que você quer:
   - Integrar o CRM da sua imobiliária ao portal
   - Enviar anúncios automaticamente
   - Receber leads em tempo real (webhook ou e‑mail estruturado)

## 2. QuintoAndar

### 2.2. Rede QuintoAndar / Parcerias
- Rede QuintoAndar (para imobiliárias e corretores parceiros):  
  https://rede.quintoandar.com.br

### 2.3. Canais de Contato (integração)

1. Cadastrar sua imobiliária / operação na **Rede QuintoAndar**.
2. Acessar a central de ajuda / suporte dentro da própria Rede.
3. Solicitar acesso ou informações sobre **“integração automática de anúncios”** ou **“integração com CRM”**.

## 3. Chaves na Mão

- Portal: https://www.chavesnamao.com.br
- Área para parceiros/anunciantes:  
  Normalmente acessível via rodapé do site, links como “Anunciar”, “Imobiliárias”, “Parceiros” ou “Planos”.

### Integração
O padrão de mercado para esse portal é usar **feed XML** em vez de uma API REST pública. O procedimento usual é:

1. Entrar na área de **parceiros** ou **planos para imobiliárias** no site.
2. Localizar menção a **“integração via XML”**, “integração com CRMs” ou “integração com portais”.
3. Usar o formulário de contato comercial informando que:
   - Você usa (ou pretende usar) um CRM próprio
   - Deseja receber o padrão de XML ou credenciais para integração.

## 4. Imovelweb / WImóveis

> Observação: existem empresas com nomes muito parecidos (“ImóvelWeb”, “ImoveisWeb”, etc.). Abaixo, os domínios amplamente usados no mercado.

- Imovelweb (portal nacional): https://www.imovelweb.com.br
- WImóveis (Centro-Oeste, DF e região – grupo ligado ao Imovelweb):  
  https://www.wimoveis.com.br

### Integração
O modelo usual é por:
- **Feed XML** para envio de imóveis
- Integração com CRMs e ERPs imobiliários

Como localizar:
1. Rodapé do site → links do tipo **“Imobiliárias”**, **“Anuncie seus imóveis”**, **“Integração”**.
2. Procure páginas com termos como **“integração com sistemas”**, **“integração com CRM”**.
3. Use os formulários dessas páginas para solicitar:
   - Padrão de XML
   - Instruções para homologar a integração

## 8. Provedores de Dados e Inteligência Imobiliária

Esses não são portais de anúncio em si, mas fontes de **dados de mercado** que podem ser integradas ao seu CRM para enriquecer análises, pricing, etc.

### 8.1. DataZAP
- Site institucional: https://www.datazap.com.br
- Foco: inteligência de mercado, índices, relatórios.

Integração:
- Normalmente feita sob contrato, com acesso a **APIs de dados** ou **bases para BI**.
- Contato via formulários de **“Fale Conosco”**, **“Produtos”** ou **“Soluções para empresas”**.

### 8.2. Índice FipeZAP
- Portal: https://www.fipezap.com.br
- Índice de preço de imóveis com cobertura nacional.

Integração:
- Não há API pública ampla e totalmente aberta no padrão “REST + chave” como em big techs; o mais comum é consumir dados via:
  - Downloads públicos de relatórios e séries históricas
  - Parcerias institucionais com FIPE/DataZAP.

### 8.3. SINAPI (Custos da Construção)
- Portal Caixa/SINAPI: procure por “SINAPI Caixa Econômica Federal” (endereço pode mudar ao longo do tempo).
- Dados: custos de materiais e índices de construção.

Integração:
- Em geral via downloads de **planilhas e arquivos abertos**, que você pode importar para um data warehouse próprio.

## 9. Plataformas de Atendimento Omnichannel / CRM com Foco em Imobiliárias

Essas não são portais de anúncio, mas sim **plataformas que já têm integrações prontas** (ou semi‑prontas) com vários portais e WhatsApp. Podem ser usadas como “ponte” entre portais e seu CRM próprio.

### 9.1. Nexloo
- Site: https://nexloo.com
- Foco: atendimento omnichannel (WhatsApp, Instagram, Facebook, etc.).

Uso típico:
- Centralizar conversas (incluindo leads de portais, quando integrados)
- Integrar com CRMs através de webhooks/API própria da Nexloo.

### 9.2. Cloud Canal – Omnichannel
- Página de omnichannel:  
  https://cloudcanal.com.br/omnichannel-cloud/

Uso típico:
- Centralizar múltiplos canais de atendimento (WhatsApp, chat, etc.)
- Fazer “ponte” entre canais e um CRM interno via integrações.

### 9.3. Plataformas Imobiliárias Especializadas (exemplos)

Estas soluções normalmente já têm **integrações homologadas** com ZAP, Viva Real, OLX, Imovelweb, etc., e permitem plugar seu próprio CRM via API:

- Jetimob: https://www.jetimob.com
- Imobzi: https://www.imobzi.com.br
- Vista Software: https://www.vistasoft.com.br
- App Facilita: https://www.appfacilita.com

Nesse cenário, você pode:
1. Usar essas plataformas como “hub” de portais.
2. Consumir a API delas a partir do seu CRM para:
   - Sincronizar imóveis
   - Centralizar leads
   - Distribuir contatos para corretores.

## 10. Estratégia Recomendada para Contato

Como muitos portais **não expõem uma API REST pública e aberta** como um Stripe ou Mercado Pago, o fluxo de integração costuma ser:

1. **Listar os portais prioritários** (ex.: ZAP, Viva Real, OLX, QuintoAndar, Imovelweb).
2. Para cada um:
   - Encontrar a página de **“Imobiliárias / Anunciar”** no rodapé.
   - Ver se há menção explícita a **“integração com sistemas”**, **“CRMs parceiros”**, **“feed XML”**.
   - Se sim, seguir o passo a passo da própria página.
   - Se não, usar o **formulário de contato** perguntando diretamente sobre:
     - Envio automático de anúncios via XML/API.
     - Recebimento automático de leads.
3. Em paralelo, decidir se você vai:
   - Integrar **direto em cada portal** (mais controle, mais trabalho); ou
   - Usar um **hub (CRM imobiliário / omnichannel)** já integrado a vários portais.

---

## 11. Dica Prática para Organização no CRM

Ao montar seu CRM/omnichannel, sugere‑se manter uma tabela ou planilha com colunas como:

- Nome do portal
- URL principal
- URL de integração / desenvolvedores
- Tipo de integração (XML, REST, Webhook, apenas via parceiro)
- Status do relacionamento (em prospecção, em homologação, ativo)
- Contato comercial
- Contato técnico

Esse arquivo de **links & contatos** serve como base viva para sua equipe saber **onde falar sobre integração** e **onde estão as documentações oficiais** de cada parceiro.
